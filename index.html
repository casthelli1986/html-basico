Conceito do projeto

Conceito geral

Este projeto começou a ser elaborado em 15 de agosto de 2023 e faz parte de meus estudos em HTML, e o objetivo é conhecer todos os princípios básicos dessa linguagem de marcação de hyper texto.

Sendo assim, observe que a aparência do projeto será muito simples, pois não usarei em nenhum momento o CSS. Se você não sabe nada sobre isso, o HTML contém tags que delimitam em uma página web o que será apresentado, mas é o CSS que faz a estilização destes componentes e dá uma aparência mais elegante ao conteúdo.

Objetivo

Então, o objetivo deste pequeno projeto é estabelecer algumas informações úteis para quem está também começando nesta área e quer aprender desde o básico tendo um material de referência. Mas este projeto será elaborado com HTML puro, e por isso, a aparência simples.

História da internet

Comunicação entre cliente e servidor

A internet, como já foi dito antes, é uma rede de computadores conectados no mundo todo. Para começar a entender como esta comunicação funciona, devemos nos familiarizar com os termos cliente e servidor. O que isso quer dizer?

Toda a internet funciona através de requisições por parte de um lado da conexão e repostas as estas requisições por parte do outro lado da conexão. Quando você acessou esta página, usou um navegador instalado em algum dispositivo eletrônico. Pode ter sido através de um computador, notebook, celular ou outro. Quando você acessou um endereço na internet, você fez uma requisição por meio do navegador.

Do outro lado da conexão está outro computador conectado à internet, 24 horas por dia. Este computador é praparado para este tipo de tarefa, e ao receber a requisição, retornou com o conteúdo do site que você acessou.

Você percebe que nesta breve explicação já podemos identificar quem foi o cliente e quem foi o servidor? Para ilustrar, imagine que você (cliente), vai até uma lanchonete e pede para um atendente (servidor) te preparar um cachorro quente. Para cumprir esta tarefa, ele tem alguns ingredientes que ele sabe juntá-los e fazer a partir deles um cachorro quente.

Bom, então, dentro desta comparação, imagine que a lanchonete em si é a hospedagem, ou seja, um estabelecimento comercial que disponibiliza servidores onde nós usuários podemos criar nossos sites e hospedar outros tipos de recursos. Cada funcionário da lanchonete é um servidor que atende às solicitações dos clientes. E o que seriam os ingredientes para montar o cachorro quente no cenário da internet?

Bom, um site é feito de vários arquivos. Podem ser uma mescla de arquivos HTML, CSS, Javascript, algumas imagens e outros arquivos como fontes, ícones e outros. Também vai depender da tecnologia utilizada para a construção do site. Mas o que importa é que o servidor tem as instruções necessárias para responder a requisição do navegador de uma forma que ele possa interpretar o que foi enviado. Este irá receber os arquivos solicitados e interpretá-los formando uma página web.

Bom e é assim que funciona a dinâmica da internet. O HTML neste caso é o arquivo estático que fica armazenado no servidor. É um dos componentes que formam a maioria dos sites que vemos na internet. Esta página que você está visualizando agora nada mais é do que um arquivo HTML puro.
